API de gestão de clientes

  • GET /api/customers - Lista todos os clientes
  • GET /api/customers/:document - Lista cliente a partir do documento (CPF ou CNPJ)
  • POST /api/customers - Cria um novo cliente
  • PUT /api/customers/:document - Altera os dados de um cliente
  • DELETE /api/customers/:document - Exclui um cliente
